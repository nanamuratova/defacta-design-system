How it's used

1 · Open in Claude Code

Open this folder in Claude Code. It auto-loads CLAUDE.md — the rules that keep every screen on-brand.

2 · Describe a screen

“Build a settings page with a profile form.” Claude reuses these components and tokens automatically.

3 · Re-theme anytime

Change a value in tokens.css and every screen updates at once — colors, type, spacing.
